Introduction

Introduction

What is data science?

It is the…

  • systematic application of scientific principles
  • to use (often large) data sets
  • to rigorously answer
  • questions of scientific or organizational interest

In this class, you will learn to carry out this process using good practices for managing data, writing and documenting code, and interpreting and presenting results.
